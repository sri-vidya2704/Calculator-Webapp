# Calculator-Webapp
A fully functional calculator built using HTML, CSS, and JavaScript.  This project contains:   DOM Manipulation, Event Handling, Keyboard Events, JavaScript Functions, Arithmetic Operations, Error Handling.

Features:

Basic arithmetic operations
Addition
Subtraction
Multiplication
Division
Interactive UI
Responsive button layout
Keyboard input support
AC (All Clear) functionality
DEL (Delete last character) functionality
Error handling for invalid expressions
Hover and button press effects

Technologies Used:

HTML5
CSS3
JavaScript (Vanilla JS)
